## üí° Pro Tips

### First-Time Users
1. Start with [QUICKSTART.md](./QUICKSTART.md)
2. Try adding a few preset signals
3. Read the full [README.md](../README.md) to understand the methodology
4. Experiment with custom signals

### Developers
1. Read [CONTRIBUTING.md](../CONTRIBUTING.md) for code standards
2. Review [PRD.md](../PRD.md) for design context
3. Check source code comments for implementation details
4. Run `npm run dev` and explore the code

### Educators
1. Review [README.md](../README.md) for educational use cases
2. Use [QUICKSTART.md](./QUICKSTART.md) for classroom setup
3. Create custom signal scenarios for students
4. Explore `src/lib/signal-presets.ts` for preset examples
