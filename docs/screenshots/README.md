Screenshots for this project need to be captured from a running instance of the application.

### To Add Screenshots:

1. **Run the application locally**:
   ```bash
   npm install
   npm run dev
   ```

2. **Follow the screenshot guide**:
   See [SCREENSHOTS.md](SCREENSHOTS.md) for detailed instructions on capturing high-quality screenshots.

3. **Place screenshots in this folder**:
   - `banner.png` (1200x630px) - Main hero image
   - `clock.png` (800x800px) - Doomsday clock close-up
   - `signals.png` (1000x800px) - Signal feed list
   - `presets.png` (900x700px) - Quick presets dialog
   - `stats.png` (1000x300px) - Statistics dashboard
   - `context.png` (400x600px) - Historical context panel (optional)

### Tips for Best Screenshots:

- Use 100% browser zoom
- Add 3-4 diverse signals for visual interest
- Capture when glow effects are at peak brightness
- Use a dark environment to maximize contrast
- Ensure text is crisp and readable
- Optimize file size after capture (<500KB each)
